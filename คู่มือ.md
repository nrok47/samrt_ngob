# 📘 คู่มือการใช้งาน Smart ติดตามกำกับ งบประมาณ ศอ.10
### ระบบ GFMIS Budget Dashboard — ปีงบประมาณ 2569

---

## 🗺️ ภาพรวม Flow การใช้งานตลอดปีงบ

```
[เริ่มปีงบ ต.ค. 2568]
       │
       ▼
1️⃣  ตั้งค่าแผนงานงบประมาณ → เพิ่มข้อมูลใน DS2 ผ่านเมนูระบบ
       │
       ▼
2️⃣  ตั้งเป้าหมายรายเดือน → Tab "เมนูระบบ" → ตั้งค่าเป้า DS3
       │
       ▼
3️⃣  รายงานผล GFMIS รายวัน → Tab "อัปเดต GFMIS รายวัน"
       │
       ▼
4️⃣  บันทึกกันเงิน/เบิกจ่าย → Tab "กันเงินรายวัน"
       │  (ทำซ้ำทุกวัน/สัปดาห์)
       ▼
5️⃣  ติดตามสถานะ → Tab "ภาพรวมผู้บริหาร"
```

---

## 📌 ขั้นตอนที่ 1 — ตั้งค่าแผนงานงบประมาณเริ่มต้นปี

> ทำครั้งเดียวตอนเริ่มต้นปีงบ ต.ค. 2568

1. เลือก **ปีงบ 2569** จาก Dropdown มุมบนขวา
2. กดปุ่ม **เมนูระบบ** (มุมบนขวา)
3. เลือก **"เพิ่มแผนงาน / รหัสงบ"** แล้วกรอกข้อมูล:
   - ชื่อแผนงาน
   - รหัสงบ (20 หลัก เช่น `21009140002001000000`)
   - ประเภทงบ: A (ดำเนินงาน) / B (บุคลากร) / C (ลงทุน)
   - งบจัดสรร (Alloc_Total) จาก GFMIS NFMA46
4. ทำซ้ำจนครบทุกแผนงาน
5. ตรวจสอบใน **Tab "รายละเอียดงบ"** ว่าครบถ้วน

> ✅ เช็ก: ยอดรวม "งบจัดสรรรวม (A+B+C)" บน Dashboard ตรงกับที่ได้รับ

---

## 📌 ขั้นตอนที่ 2 — อัปเดต GFMIS รายวัน (ทุกวันที่มีการเบิกจ่าย)

> ทำทุกวัน/ทุกสัปดาห์ โดยดูจากรายงาน NFMA46 ใน GFMIS

1. เลือก **ปีงบ 2569**
2. ไปที่ Tab **"อัปเดต GFMIS รายวัน"**
3. กด **"โหลดข้อมูล GF_Input"**
   - ระบบจะแสดงรายการแผนงานทั้งหมดของปี 2569
   - คอลัมน์สีเทา (PO/เบิก เมื่อวาน) = ค่าจากวันก่อน
4. กรอก **PO วันนี้** (สีส้ม ✏) และ **เบิก วันนี้** (สีเขียว ✏) สำหรับแผนงานที่มีการเคลื่อนไหว
   - ระบบคำนวณ **คงเหลือ** ให้อัตโนมัติ
5. กด **"บันทึกและอัปเดต DS2"**
   - ยืนยัน → ระบบอัปเดต DS2_MasterBudget ทันที
   - Dashboard จะรีเฟรชโดยอัตโนมัติ

> ⚠️ ข้อควรระวัง: กรอกเป็นยอด **สะสมทั้งหมด** (ไม่ใช่ยอดเพิ่มรายวัน) ตรงกับตัวเลขใน GFMIS

---

## 📌 ขั้นตอนที่ 3 — บันทึกกันเงินรายวัน (ทุกครั้งที่มีการกันเงิน)

> บันทึกการกันเงินโครงการ/กิจกรรมแต่ละรายการ

1. ไปที่ Tab **"กันเงินรายวัน"**
2. กดปุ่ม **"+ เพิ่มกันเงิน"**
3. กรอก:
   - วันที่กันเงิน
   - กลุ่มงาน / Admin_Line (สายบริหาร)
   - กิจกรรม / ชื่อโครงการ
   - ยอดเงิน
   - ประเภทงบ (A/B/C)
4. กด **"บันทึก"**

---

## 📌 ขั้นตอนที่ 4 — ติดตามสถานะ

| Tab | ใช้ดู |
|---|---|
| ภาพรวมผู้บริหาร | KPI รวม, % เบิกจ่าย, ग्राฟ |
| รายกลุ่มงาน | แยกตามสาย/กลุ่มงาน |
| รายละเอียดงบ | ค้นหา/กรอง แต่ละแผนงาน |
| สายบริหาร | สรุปยอดรายสาย |
| รอบโอนงบ | ประวัติการโอนงบระหว่างแผนงาน |
| ประวัติการใช้งาน | Audit log ทุกการเปลี่ยนแปลง |

---

## 🔁 Flow รายวัน (Routine)

```
ทุกวัน (ที่มีข้อมูล GFMIS ใหม่):
  1. เปิดรายงาน NFMA46 จาก GFMIS
  2. ไป Tab "อัปเดต GFMIS รายวัน"
  3. โหลด → กรอก PO+เบิก (สะสม) → บันทึก
  4. ตรวจ Dashboard ว่า % เบิกจ่ายถูกต้อง
```

## ❓ FAQ

**Q: ตัวเลขบน Dashboard ไม่อัปเดต?**
→ กดปุ่ม **"รีเฟรช"** มุมบนขวา หรือเลือกปีงบใหม่แล้วเลือกกลับ

**Q: แผนงานหายไปใน Tab อัปเดต GFMIS?**
→ ตรวจสอบว่า `Include_In_Budget = TRUE` และ `Fiscal_Year = 2569` ใน DS2

**Q: ยอดรวม KPI ต่างจาก GFMIS?**
→ "เบิก GFMIS (NFMA46)" รวมงบสวัสดิการ/บำนาญด้วย (Type D) ถ้าไม่เท่ากันให้ตรวจรหัสงบในแผนงาน

**Q: ต้องการย้อนดูปีเก่า?**
→ เปลี่ยน Dropdown ปีงบมุมบนขวา — ข้อมูลแยกปีงบโดยอัตโนมัติ
