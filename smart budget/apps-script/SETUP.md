# คู่มือเชื่อม Dashboard กับ Google Sheet (Real-time)

> สำหรับทีม smart budget · ศูนย์อนามัยที่ 10 อุบลราชธานี

## ภาพรวม

```
┌─────────────────────┐    ┌──────────────────┐    ┌─────────────────────┐
│  Google Sheet       │    │  Apps Script     │    │  Dashboard (HTML)   │
│  (ทีมกรอกทุกวัน)    │ →  │  (Web App / API) │ →  │  (แสดงผล / refresh) │
│  GFMIS + การเงิน    │    │  Code.gs         │    │  index.html         │
└─────────────────────┘    └──────────────────┘    └─────────────────────┘
```

ทีมยังคงกรอกข้อมูลใน Google Sheet ตามปกติ → Apps Script เปลี่ยนชีตเป็น JSON API → Dashboard ดึงไปแสดง refresh เองทุก 5 นาที

## ขั้นตอนที่ 2 · ติดตั้ง Apps Script

1. เปิด Google Sheet → เมนู **Extensions → Apps Script**
2. ลบโค้ดที่ขึ้นมาเริ่มต้นออกหมด
3. เปิดไฟล์ `apps-script/Code.gs` (ในโปรเจกต์นี้) → คัดลอกทั้งหมด → วางใน Apps Script
4. แก้ค่าใน `CONFIG` ให้ชื่อตรงกับ tab ในชีต (ถ้าตั้งชื่อตามขั้นตอนที่ 1 ไม่ต้องแก้)
5. กด **บันทึก** (ไอคอนดิสก์) → ตั้งชื่อโปรเจกต์ `smart-budget-api`
6. กดปุ่ม **Run** เลือกฟังก์ชัน `testRead` → อนุญาตสิทธิ์ → ดูผลที่ **View → Logs** ว่าออกมาเป็น JSON หรือไม่

### Deploy เป็น Web App

1. กด **Deploy → New deployment**
2. ที่ icon เกียร์ เลือก **Web app**
3. กรอก:
   - Description: `smart-budget API v1`
   - Execute as: **Me (your email)**
   - Who has access: **Anyone** ⚠️ (จำเป็น — แต่ URL ยาวจน guess ไม่ได้ ถือว่า private)
4. กด **Deploy** → คัดลอก **Web app URL** เก็บไว้

ทดสอบ: เปิด URL นั้นบน browser → ต้องเห็น JSON ขึ้นมา

## ขั้นตอนที่ 3 · เชื่อม Dashboard

แก้ไฟล์ `Budget Report Realtime Dashboard.html` ด้วย URL ที่ได้:

```js
// หาบรรทัดนี้ใน <script> แล้วใส่ URL:
const API_URL = 'https://script.google.com/macros/s/XXXXXXXXX/exec';
```

ที่ผมทำให้แล้ว: dashboard จะ:
- โหลดข้อมูลครั้งแรกเมื่อเปิดหน้า
- refresh เองทุก **5 นาที**
- ถ้า fetch ไม่สำเร็จ → ใช้ข้อมูล demo สำรอง + แสดงสถานะ "offline"

---

## ขั้นตอนที่ 4 · update ข้อมูลรายวัน

ทีมแค่กรอกตัวเลขใหม่ใน Sheet → dashboard จะดึงตอน refresh รอบถัดไปเอง

**Tips:**
- ใช้ Google Form ก็ได้ ให้ทีมกรอกผ่านมือถือ → ส่งเข้า Activity tab
- ตั้ง trigger ใน Apps Script ให้ส่ง LINE Notify ตอนข้อมูลครบ (ถ้าต้องการ)
- ถ้าจะแก้ schema → ต้อง redeploy Apps Script (Deploy → Manage deployments → แก้ version)

## Troubleshooting

| อาการ                              | วิธีแก้                                            |
|-----------------------------------|----------------------------------------------------|
| Dashboard ขึ้น "offline"           | เช็ค URL · เช็ค "Who has access" ตั้ง Anyone หรือยัง |
| JSON ออกแต่ค่าเป็น 0               | เช็คชื่อ tab ใน CONFIG ตรงกับชีตจริงไหม              |
| ค่าผิดทศนิยม                       | ในชีตให้เป็น Number format ไม่ใช่ Text             |
| ไม่ refresh                       | F5 หรือรอครบ 5 นาที · เปิด console ดู error        |
| แก้ Apps Script แล้วไม่อัปเดต      | Deploy → Manage deployments → Edit → New version  |
